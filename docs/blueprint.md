# **App Name**: BharatPulse Sports

## Core Features:

- Live Event Dashboard: A dynamic home screen displaying real-time upcoming sports events featuring Indian athletes, filterable by sport type.
- Global Sport Search: Unified search and filter system for finding events by name, location, or level (International vs. National).
- Athlete Briefing Tool: A generative AI-powered tool that provides a summary of the current form and previous performances of the participating Indian athletes.
- Event Detail Matrix: Detailed view for specific events, providing comprehensive data including venue, city, levels, and external official URLs.
- Central Sports Repository: Integrated system utilizing Firestore to store and synchronize sports categories and event data.
- Participation Tracker: Specific list management within event pages to show all Indian athletes and teams qualified for the competition.

## Style Guidelines:

- Primary Color: Energetic Saffron (#FF833E) reflecting national pride and athletic vigor.
- Background Color: Deep Onyx (#171412), a heavily desaturated version of the primary hue for a premium dark mode experience.
- Accent Color: Vibrant Crimson (#F62B44), an analogous hue for highlighting interactive elements and calls to action.
- Headline Font: 'Space Grotesk' for a high-tech, scoreboard-inspired aesthetic; Body Font: 'Inter' for clear readability in data-heavy lists.
- Clean, thin-line geometric icons that look precise and professional on a dark interface.
- Responsive card-based design with an airy grid system, optimized for rapid scanning on mobile and desktop.
- Smooth slide-and-fade transitions when switching between sport filters or navigating to event details.
